# 1/ Les formulaires

## A/ Page de connexion

En partant du fichier `login.html` :

```html
<!DOCTYPE html>
<html lang="fr">
    <head>
        <meta charset="UTF-8">
        <title>Connexion</title>
    </head>
    <body>
        <h1>Connexion</h1>
    </body>
</html>
```
Créez un formulaire de connexion avec les champs suivants :
- Un champ pour le nom d'utilisateur (input de type text).
- Un champ pour le mot de passe (input de type password).
- Un bouton de soumission.
- Ajoutez les contraintes suivantes :
- Les champs doivent avoir des labels associés.
- Le champ du nom d'utilisateur doit être requis.
- Le champ du mot de passe doit être requis et avoir un minimum de 6 caractères.

## B/ Page d'inscription

Dans un fichier register.html, créez un formulaire d'inscription avec les champs suivants :

- Un champ pour le nom complet (input de type text).
- Un champ pour l'adresse email (input de type email).
- Un champ pour le mot de passe (input de type password).
- Un champ pour la confirmation du mot de passe (input de type password).
- Un champ pour la date de naissance (input de type date).
- Un champ pour choisir le genre (input de type select avec les options "Homme", "Femme", "Autre", et une option "-- Sélectionnez votre genre --" par défaut non sélectionnable).
- Un champ pour sélectionner son niveau d'expérience en développement web (input de type radio avec les options "Débutant", "Intermédiaire", "Avancé").
- Un champ pour choisir les langages de programmation maîtrisés (input de type select avec l&apos;attribut multiple) avec les options "HTML", "CSS", "JavaScript", "Java" et une option par défaut non sélectionnable.
- Un champ pour télécharger une photo de profil (input de type file).
- Une case à cocher pour accepter les conditions générales (input de type checkbox).
- Un bouton de soumission.

Ajoutez les contraintes suivantes :

- Tous les champs, à l'exception de la photo de profil, sont obligatoires.
- Le champ du mot de passe doit être requis et avoir un minimum de 8 caractères.

## C/ Pour aller plus loin
### 1/ Bootstrap
Modifiez le &lt;head&gt; des fichiers login.html et register.html pour y ajouter :

```html
<link href="https://cdn.jsdelivr.net/npm/bootstrap@5.3.3/dist/css/bootstrap.min.css" rel="stylesheet" integrity="sha384-QWTKZyjpPEjISv5WaRU9OFeRpok6YctnYmDr5pNlyT2bRjXh0JMhjY6hW+ALEwIH" crossorigin="anonymous">
```
Utilisez les classes fournies par Bootstrap pour améliorer l'esthétique de la page. Nous verrons le fonctionnement des classes .mb lors du TP sur Bootstrap.

### 2/ L'autocomplete
Ajoutez l'utilisation de l'attribut autocomplete dans les formulaires pour améliorer l'expérience utilisateur et faciliter la saisie des informations.
